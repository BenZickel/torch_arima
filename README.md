# torch_arima

ARIMA time series implementation in PyTorch implemented as a torch `torch.nn.Module` called `ARIMA`.

## Examples

### Maximum Likelihood Estimator

```
python -m ARIMA.examples.mle
```

### Bayesian Estimator

```
python -m ARIMA.examples.bayesian
```

## Tests

```
python -m ARIMA
```
